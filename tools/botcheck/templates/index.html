Bot check

A live score of how much your browser looks like a human vs. an automated bot. We combine what your browser reports (via JavaScript) with what your connection actually shows the server (HTTP headers, IP reputation), then cross-check the two: the disagreements are what give bots away. One thing is kept: a hash of your browser's stable fingerprint plus the IP it arrived from, held in a rolling 30-day corpus that catches the same fingerprint reused across many IPs. The check runs the moment this page loads.

What this checks, and what it doesn't

The score comes from three things: your browser's own JavaScript fingerprint, your HTTP request headers, and the reputation of your IP address (datacenter, VPN, proxy, Tor). The strongest checks are the cross-checks: where what your browser claims disagrees with what your connection actually shows.

It does not use behavioral tracking, mouse or keystroke analysis, or machine learning: every deduction is a fixed, explainable rule you can read below. The one thing stored is a rolling 30-day fingerprint corpus — a one-way hash of the stable fingerprint plus the IP that presented it — which a single rule uses to catch the same exact fingerprint arriving from many IPs (the scraping-farm tell). Sightings expire automatically; nothing else is kept.

A low score means "looks automated", not "is malicious". Privacy-conscious and VPN users can score suspicious by design, and false positives are expected: this is a transparent self-test, not a gate that blocks anything.

Using this from the terminal

Every view also speaks JSON — browsers get the page, everything else gets JSON.

# server-only score of your request (no JS signals)
curl https://botcheck.corpberry.com

# score a fingerprint you collected yourself
curl -X POST https://botcheck.corpberry.com/check \
  -H 'Content-Type: application/json' -d '{"webdriver":true}'

The POST /check JSON response echoes your fingerprint back as clientPayload — the same raw dump the page shows under "raw fingerprint" — so you can diff exactly what was scored.
