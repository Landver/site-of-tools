Using this from the terminal

Every view also speaks JSON — browsers get the page, everything else gets JSON.

# your IP + geolocation + connection (JSON)
curl https://ip.corpberry.com

# look up any IP — geolocation + proxy/VPN (JSON)
curl 'https://ip.corpberry.com/?ip=8.8.8.8'

# subnet / CIDR calculator (JSON)
curl 'https://ip.corpberry.com/cidr?cidr=10.0.0.0/8'
